ИНКСТРУКЦИЯ ПО СБОРКЕ gulp.
Для того чтобы запустить gulp-сборку необходимо задать следующие команды 

Команды производятся через терминал в Visual Studio Code

1каманда-------    npm i     ----------- 
Далее начинается загрузка необходимых файлов для работы этой сборке.


2каманда-------    gulp        -----------
Запускает эту сборку, в котором можно делать нужный сайт.
То есть, работать над проектом «scss, index.html, main.js, ».

3каманда-------     gulp build       -----------
По завершений  над проектом запускаем эту команду для того чтобы готовый-проект был готов. 
Создается  папка dist в котором содержатся файлы с проектом тесть наш сайт «dist, css, fons, js, index.html».   



git залить на  gitHab
1) добавить  в изминения каманд  : git add . 
2) проверяем статус : git status
3) камнтарий к изменению: git commit -m ""
4) залить все на сервер gitHab : git push
